For help with ViralForge, email philyu2023@qq.com.

Restore Purchases

Open ViralForge, go to Settings or the Pro tab, then tap Restore Purchases. Make sure you are signed in with the same Apple ID used to purchase the subscription.

Generation Issues

If generation fails, check your network connection and try again. If backend mode is enabled, confirm the backend URL is reachable.

Data Deletion

For backend-synced data deletion, email your in-app User ID to philyu2023@qq.com. Local-only data can be removed by deleting the app from the device.
