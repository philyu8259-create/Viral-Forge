Support / 支持

For help with ViralForge, email support@viralforge.app.

如需 ViralForge 使用帮助，请发送邮件至 support@viralforge.app。

Restore Purchases / 恢复购买

Open ViralForge, go to Settings or the Pro tab, then tap Restore Purchases. Make sure you are signed in with the same Apple ID used to purchase the subscription.

打开 ViralForge，进入设置页或会员页，点击“恢复购买”。请确认当前 Apple ID 与购买订阅时一致。

Generation Issues / 生成问题

If generation fails, check your network connection and try again. If backend mode is enabled, confirm the backend URL is reachable.

如生成失败，请检查网络并重试。如开启后端模式，请确认后端地址可访问。

Data Deletion / 数据删除

For backend-synced data deletion, email your in-app User ID to support@viralforge.app. Local-only data can be removed by deleting the app from the device.

如需删除后端同步数据，请将 App 内用户 ID 发送至 support@viralforge.app。仅保存在本机的数据可通过卸载 App 删除。
